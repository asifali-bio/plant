# Code of Conduct

## Our Commitment

This project is committed to fostering an open, respectful, and collaborative environment.  
We expect all participants to engage in good faith and contribute constructively.

## Expected Behavior

- Be respectful and considerate in all interactions
- Provide constructive feedback and engage in scientific discussion in good faith
- Assume positive intent, especially in early-stage contributions

## Unacceptable Behavior

This includes, but is not limited to:

- Harassment, discrimination, or personal attacks
- Dismissive or non-constructive criticism
- Disruptive behavior that undermines collaboration

## Scope

This Code of Conduct applies to all project spaces, including issues, pull requests, discussions, and related communications.

## Enforcement

Project maintainers reserve the right to remove, edit, or reject contributions and to restrict participation for behavior that violates these guidelines.

## Reporting

If you experience or observe unacceptable behavior, please contact Asif Ali.
